How to Use Interactive Strawmap

Drag and Drop

Drag roadmap items between fork columns to explore different timeline scenarios.

Dependency Warnings

When you move an item that breaks dependencies, you'll see warnings and auto-suggestions.

Visual Indicators

  • Critical Path: Items that block the most others
  • Fork Complexity: Number and difficulty of items per fork
  • Dependency Arrows: Technical prerequisites between items

Controls

  • Reset: Return to original strawmap configuration
  • Undo/Redo: Navigate through your changes
  • Export: Save your modified roadmap configuration
